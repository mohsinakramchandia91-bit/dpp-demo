# Digital Product Passport (DPP)

Enterprise-ready Digital Product Passport system.

## Features
- Buyer verification
- Public product passport
- Company analytics dashboard
- EU-ready compliance concept

## Tech Stack
- Frontend: Static HTML (GitHub Pages)
- Backend: Flask API
- Charts: Chart.js

## Run Backend (Local)
cd backend
pip install -r requirements.txt
python app.py

## Frontend
Open frontend/index.html or deploy on GitHub Pages.
